UnLaSE: Towards Unsupervisd Latent Representations for Livestock Image Segmentation
Veeraraju Elluru, Ahmed Sallam, Anderson Alves, Tiago Bresolin

USPLF 2025
paper / code / program page

We propose the implementation of a novel self-supervised algorithm for generating precise cattle masks, at scale. Existing works that use SSL were primarily based on contrastive learning which necessitates positive and negative samples. Our work reduces this need drastically and can generate highly accurate masks across cattle breeds and environmental conditions.
